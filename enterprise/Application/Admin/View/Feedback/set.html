添加招聘信息
前台调用方法： 可在栏目配置中添加，访问地址为"http://localhost/feedback/"
防刷新时间： 秒，同一IP2次提交的最小间接时间
敏感字符过滤： 多个字符请用"|"隔开
信息接收方式： 邮件接受 写入数据库从后台读取 发送邮件并写入数据
反馈邮件接收邮箱： 多个邮箱请用"|"隔开
自动回复： 是否自动向提交反馈的客户回复邮件
email字段名： 用于填写email的反馈表单字段中文名称，如Email、电子邮件等，英文区分大小写
回复邮件标题： 自动回复邮件的标题
回复邮件内容： 支持HTML语言
